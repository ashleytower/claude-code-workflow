## 1. THE LAYER STACK: Anatomy of a Professional Video Frame

A professional short-form video frame is built from distinct visual layers stacked in Z-order:

```
LAYER 7 (top):   Animated captions / subtitles (word-by-word highlight)
LAYER 6:         CTA elements -- buttons, swipe-up prompts, end cards
LAYER 5:         Primary headline text with background treatment
LAYER 4:         Graphic overlays -- bars, shapes, badges, accent lines
LAYER 3:         Gradient scrim / vignette / color wash
LAYER 2:         Color grade / LUT applied to footage
LAYER 1 (base):  Raw footage or background image
```

Each layer has a specific purpose. The $50K agency difference is in how precisely these layers interact.

## 2. TYPOGRAPHY: The Design Backbone

### Font Selection Rules
- **Primary headline**: Bold/Black weight condensed sans-serif (Bebas Neue, Oswald, Montserrat ExtraBold, Fira Sans Condensed Bold)
- **Supporting text**: Medium weight of the same family OR a contrasting clean sans-serif (Proxima Nova, Inter, DM Sans)
- **Accent/label text**: Light or regular weight, all-caps, wide letter-spacing
- Avoid serif fonts -- they break down after video compression on mobile
- Avoid thin-stroke fonts -- illegible at small sizes on phone screens
- Limit to ONE font family; use weight/size/color for hierarchy, not multiple typefaces

### Font Sizing for Mobile (1080x1920 canvas)
- **Hero headline**: 80-120px (fills ~40-60% of frame width)
- **Supporting subhead**: 48-64px
- **Body/caption text**: 36-48px
- **Fine print/source attribution**: 24-32px
- Rule: Two short lines beats one long line on mobile
- Hook text: 5-9 words maximum for instant legibility

### Text Positioning
- **Upper third placement** is safest from platform UI overlays
- **Center-frame** for maximum impact hero statements
- **Lower-left** for classic lower-third name/title cards
- Left-aligned stacking with name at ~2x the size of subtitle text
- NEVER place critical text in the bottom 320px (TikTok captions) or right 120px (engagement buttons)

### Text Contrast Techniques (ranked by sophistication)
1. **Gradient scrim**: Semi-transparent gradient (40% black to transparent) behind text area -- most professional
2. **Feathered dark oval**: Black oval behind text, 10-25% opacity, soft light or overlay blending mode
3. **Solid color bar**: Opaque or semi-transparent rectangle behind text (pill shape, rounded corners)
4. **Local blur**: Gaussian blur applied to background area behind text
5. **Text shadow/outline**: Drop shadow or stroke around letterforms -- least sophisticated
6. **Vignette darkening**: Frame edges darken 15-25% toward text area
7. **Curves adjustment**: Targeted darkening of specific tonal range where text sits

### Animation Styles
- **Pop-in**: Keywords punch onto screen (for emphasis)
- **Slide-in**: Sequential text slides from edge (for steps/lists)
- **Highlight bar**: Animated colored bar sweeps behind key phrases
- **Word-by-word reveal**: Synced to voiceover beat
- Smooth easing for calm content; sharper cuts and bouncier curves for high-energy
- Every text animation should align with audio/musical beat

## 3. GRAPHIC OVERLAY PATTERNS

### Common Overlay Elements
- **Gradient bars**: Horizontal or vertical color strips, 5-15% opacity, brand-colored
- **Accent lines**: 2-4px colored lines framing text or dividing sections
- **Pill-shaped badges**: Rounded rectangles (4-12px corner radius) containing labels ("NEW", "LIMITED", "50% OFF")
- **Geometric shapes**: Circles, triangles, or diagonal cuts that frame subjects or create visual rhythm
- **Underline accents**: Animated brush strokes or color bars beneath key words
- **Progress bars/timers**: Visual urgency elements
- **Corner brackets**: L-shaped marks in frame corners suggesting "focus" or "capture"

### Shape Language
- **Rounded rectangles** (8-12px radius): Modern, approachable, SaaS/DTC
- **Sharp rectangles**: Editorial, authoritative, news-style
- **Circles/pills**: Friendly, badge-like, social-native
- **Diagonal cuts**: Dynamic, energetic, sports/fitness
- **No border/floating**: Minimalist, luxury, high-end

### Professional Overlay Composition Rules
- "Layer with Purpose" -- overlays should emphasize key moments, never clutter
- Multiply/duplicate objects for depth and visual complexity
- Use repetition of shapes for consistency and reinforced messaging
- Maintain visual rhythm -- varied repetition that guides the eye
- Brand colors in graphic elements, not random choices

## 4. PAIN-POINT / HOOK SCENE DESIGN

### The 0.5-Second Rule
Scroll speed has increased 41% since 2020. You have 0.5 seconds to arrest attention. The design must work before the brain processes the content.

### Visual Hook Design Patterns

**Pattern 1: Bold Text Slam**
- Large condensed bold text (100-120px) fills upper 2/3 of frame
- Dark gradient scrim behind text (40-60% opacity)
- Background footage slightly desaturated to push text forward
- Text says the pain point directly: "TIRED OF [PROBLEM]?"

**Pattern 2: Split Frame**
- Left half: problem visual (desaturated, red-tinted)
- Right half: solution visual (saturated, warm)
- Dividing line can be diagonal or animated
- Text overlay bridges both halves

**Pattern 3: Post-It / Card Reveal**
- Physical card/note covers part of frame
- Handwritten-style text on card states the question
- Card peels away to reveal answer/product
- Creates curiosity gap through visual obstruction

**Pattern 4: Search Bar Hook**
- Animated Google/TikTok search bar appears
- Text types out common user query
- Transitions to product/solution reveal
- Mimics native platform behavior

**Pattern 5: Comment/Chat Simulation**
- Fake iMessage or comment thread overlay
- Casual language, emojis, alternating speakers
- Feels like peer-to-peer conversation
- Blends with organic feed content

**Pattern 6: Color Block Statement**
- Solid colored rectangle (brand color) fills 40-60% of frame
- White or contrasting bold text inside the block
- Remaining frame shows product/person
- High contrast demands attention

**Pattern 7: Red X Problem Callout**
- Product close-up in black and white
- Large red "X" mark overlaid
- Signals "this is wrong / this doesn't work"
- Immediately followed by solution frame in full color

### Timing
- Hook visual design must register in first 3 seconds
- Text overlay in hook: 64% lift in conversion rate vs. no text
- Product shown in action: 175% increase in conversion rate

## 5. PLATFORM-SPECIFIC SAFE ZONES (1080x1920 canvas)

### TikTok
- Top margin: 108px (status bar, search)
- Bottom margin: 320px (captions, progress bar)
- Left margin: 60px
- Right margin: 120px (engagement buttons)
- **Usable safe zone: 900 x 1492px centered**

### Instagram Reels
- Top margin: 220px
- Bottom margin: 420px
- Right margin: 170px
- **Text safe area: 1010 x 1280px**
- Feed view safe zone: 1080 x 1080px centered (for grid thumbnail)

### YouTube Shorts
- Top margin: 140px
- Bottom margin: 270px
- Left margin: 70px
- Right margin: 190px

### Universal Rule
- Design for the central 70-80% of the frame for text and CTAs
- Let imagery/background fill outer edges
- Place key text in upper third -- safest across all platforms

## 6. MOTION GRAPHICS TEMPLATE (MOGRT) DESIGN PATTERNS

### Reusable Component Types
1. **Lower thirds**: Name + title, left-aligned stack, animated in/out
2. **Title cards**: Full-screen text moments between footage
3. **Transitions**: Wipes, reveals, shape transitions between scenes
4. **Call-to-action end cards**: Product + CTA button + urgency text
5. **Social proof overlays**: Star ratings, review quotes, user counts
6. **Stat/number callouts**: Animated counters with source attribution
7. **Progress/timeline indicators**: Visual pacing elements
8. **Brand bumpers**: Logo reveal animations (2-3 seconds)

### Design Principles for Reusable Templates
- Build with essential properties panel for easy customization
- Allow media replacement (swap images/video without rebuilding)
- Use consistent color tokens that can be globally updated
- Build at 1080x1920 (9:16) as primary, with 1080x1080 (1:1) variant
- Animate with simple keyframes: position, scale, opacity, rotation
- Standard timing: 0.3-0.5s ease-in for entrances, 0.2-0.3s for exits
- Keep graphic elements within safe zones by default

### Modular Component Architecture
- **Header module**: Hook text + gradient scrim
- **Body module**: Supporting text + accent graphics
- **Evidence module**: Social proof, stats, before/after
- **CTA module**: Button graphic + urgency text + brand element
- Each module is independently animatable and repositionable

## 7. COLOR TREATMENT / GRADING

### Cinematic Color Grade Styles for Marketing

**Teal & Orange** (most popular)
- Push shadows toward teal/cyan, highlights toward orange/warm
- Keeps skin tones flattering while creating maximum contrast
- Best for: people-centric ads, lifestyle content
- Research: 30% higher view-through rates vs. flat grades

**Bleach Bypass / Desaturated**
- High contrast, reduced saturation
- Gritty, editorial, serious tone
- Best for: pain-point scenes, problem statements, "before" states

**High Key Saturated**
- Bright, vivid, boosted colors
- Energetic, joyful, youthful
- Best for: product reveals, solution states, "after" transformations

**Vintage / Warm Wash**
- Lifted blacks, warm color shift, reduced contrast
- Nostalgic, authentic, approachable
- Best for: UGC-style content, testimonials, story moments

**Moody / Cool Tone**
- Deep shadows, blue-shifted midtones, low saturation
- Mysterious, premium, sophisticated
- Best for: luxury/premium brands, tech products

### Color Grade Application Strategy
- Use DESATURATED/COOL grades for problem/pain-point scenes
- Switch to WARM/SATURATED grades for solution/product scenes
- The contrast between "before" and "after" color grades amplifies the transformation narrative
- Apply LUTs (3D .cube files) for consistency across all footage in a campaign
- Film grain / texture overlays at 5-15% opacity add perceived production value

### 2026 Trend: Imperfect Aesthetics
- Film grain, VHS overlays, torn paper textures
- Moving AWAY from hyper-polished AI-slick aesthetic
- Handmade feel signals authenticity
- Noise/grain at subtle levels (5-10%) adds "realness"

## 8. COMPOSITION PRINCIPLES (AGENCY-LEVEL)

### The Six Design Principles for Video Frames
1. **Balance**: Distribute visual weight across the frame -- text on one side, visual on the other
2. **Unity**: Consistent color palette, synchronized movement, cohesive brand voice
3. **Contrast**: Differentiated colors, sizes, weights to create clear hierarchy
4. **Repetition**: Repeat shapes and motifs across scenes for brand recognition
5. **Pattern**: Multiple elements repeated to heighten engagement
6. **Rhythm**: Varied repetition that guides the viewer's eye through the frame

### Shot + Overlay Integration
- **Wide shot**: Overlay text in upper third, keep lower 2/3 for scene
- **Medium shot**: Text beside or above subject, accent line between
- **Close-up**: Minimal text, maximum product/face impact, small label only
- **Dynamic/tracking**: Text tracks with movement or stays locked to frame position

### The Native Aesthetic Imperative
- High-gloss studio-shot ads see LOWER engagement on TikTok/Reels
- Design should feel platform-native, not TV-commercial polished
- Casual, user-generated styling outperforms corporate aesthetic
- But the STRUCTURE (layers, hierarchy, timing) remains professional

## SUMMARY: What Makes a $50K Frame vs. a Template

| Element | Template-Level | Agency-Level |
|---------|---------------|--------------|
| Text contrast | Drop shadow or stroke | Gradient scrim + local blur + color-matched accent |
| Typography | Random bold font | Condensed sans-serif system with 3-4 weight hierarchy |
| Color grade | Instagram filter | Intentional LUT matching brand mood + scene narrative |
| Overlays | Static rectangle | Animated branded shapes with consistent design language |
| Composition | Text centered on footage | Deliberate layer stack with safe-zone awareness |
| Animation | Fade in/out | Beat-synced kinetic typography with eased motion curves |
| Brand system | Logo slapped on | Modular component library with color tokens |
| Platform fit | One size exported | Platform-specific safe zones, native aesthetic |

The core insight: professional video frame design is a SYSTEM, not a series of one-off choices. Every element -- font weight, gradient opacity, accent color, animation timing -- follows rules that create visual consistency across an entire campaign.
